# HTTP Resolver

> HTML import resolver for the http:// and https:// schemes.

Resolves HTML imports using the `http://` and `https://` schemes.

## Install

```
npm i trucks-resolver-file --save-dev
```

For the command line interface see [trucks-cli][].
