# Name

board - compiles skatejs components

# Synopsis

```
[flags] [options] [files...]
```

```zsh
:file:_files -g '*.html'
```

# Description

Compiles skatejs components.

# Options

+ `-c, --css [file]` Stylesheet output file.
+ `-j, --js [file]` Javascript output file.
+ `-h, --help` Display help and exit
+ `--version` Print the version and exit
